# 📧 Mail Mate Extension

Mail Mate is a smart email assistant that automatically generates replies by understanding the sender's message. It combines a browser extension with a powerful backend to help users respond quickly and efficiently.

---

## 🚀 Features

* ✉️ **Automatic Email Reply Generation**
  Reads incoming emails and generates contextual replies.

* 🧠 **Smart Message Understanding**
  Uses backend processing to analyze the sender’s intent.

* 🌐 **Browser Extension Integration**
  Works directly inside your email interface for seamless usage.

* ⚡ **Fast & Responsive UI**
  Built with React for a smooth user experience.

* 🔌 **Full-Stack Architecture**
  Backend powered by Spring Boot for robust processing.

## ⚙️ Installation & Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/mail-mate.git
cd mail-mate
```

---

### 2. Backend Setup (Spring Boot)

```
cd backend
./mvnw spring-boot:run
```

Or if using Maven:

```
mvn spring-boot:run
```

---

### 3. Frontend Setup (React)

```
cd frontend
npm install
npm start
```

---

### 4. Load Extension in Browser

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer Mode**
3. Click **Load unpacked**
4. Select the `extension/` folder

---

## 🔄 How It Works

1. User opens an email
2. Extension reads the email content
3. Sends data to Spring Boot backend
4. Backend processes and generates a reply
5. Suggested reply is shown to the user
